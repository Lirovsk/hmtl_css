Então até o momento eu vou essa maneira de divisão de conteúdo. Importante salientar uma coisa: o que eu escrevi no arquivo css para estilizar no formate que você está vendo, sem enrolação: Eu ainda não escolhi uma fonte para o site, to usando o padrão do navegador. Eu apenas usei o normalize, defini a cor de fundo (é rgb(51, 51, 51)), a cor do texto (whitesmoke), o tamanho do texto ta no automático e as outras cobnfigurações eu alterei para cada container em específico. Eu também decidi usar como medida padrão o 'rem', que é uma unidade de medida relativa ao tamanho da fonte do elemento raiz (html). Isso significa que, se você aumentar ou diminuir o tamanho da fonte do elemento raiz, todos os elementos que usam 'rem' também serão afetados. Isso é útil para tornar o layout mais responsivo e acessível.

Uma coisa muito importante de salientar é dizer que eu to usando o método 'mobile firrt', então eu começo o desenvolvimento do site com um tamanho de tela de celular, depois de tudo pronto que eu vou alterar as configurações para cada tamanho de tela.
